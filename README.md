## 기능
* ID, password, 닉네임, 이메일, 전화번호 입력으로 간단한 회원가입 기능
* 목록 선택을 통해 확인하길 원하는 내용 확인 기능
* sorting을 통해 최신순, 추천순, 과거순 정렬 기능
* 글을 작성하고 사이트에 포스팅하는 기능 (card 형식)
* 이미지 업로드 및 업데이트 기능
* 목록에서 게시물 선택시 사용자에게 게시물 내용을 확인하는 기능(modal)
* 자신의 게시물 편집 및 삭제 기능
